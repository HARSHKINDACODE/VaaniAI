VaaniAI is a futuristic multilingual AI voice assistant designed specifically for India’s languages, accents, and conversational culture.

Built with a cinematic AI-first interface, VaaniAI combines real-time voice interaction, multilingual understanding, emotional intelligence, and personalized memory into a next-generation assistant experience.

✨ Features:
• Premium futuristic UI/UX
• Multilingual support for Indian languages
• AI Memory System
• Voice Sessions & Real-time Interaction
• Smart Quick Actions Overlay
• Personalized AI behavior
• Emotion-aware conversational design
• Glassmorphism + cinematic animations
• Responsive modern frontend architecture

Tech Stack:
• React.js
• Vite
• TailwindCSS
• Framer Motion
• Lucide React

Vision:
To build a human-centered AI assistant that truly understands India’s diverse voices, languages, and emotions.
